This is a scratch based logic progect that I created about small robot named Gary who has crash landed in a prehistoric forest.
He must gather enough energy to power up and escape before a nearby tyrannosaurus makes a meal out of him.

Controls: 

Gary will start opposite of the tyrannosaur, you must evade the tyrannosaur while gathering energy. As you gather energy and evade the tyrannosaur they will increase
in speed in frustration.

Control Gary with the arrow keys on your Keyboard.

Key Features:

1) This project features a scaling difficulty mechanic that increases as you progress. I built a custom block for the predator that uses real time variable monitoring
to increase thier speed as Gary's power level grows.
2) Used custom blocks to create animations with movement of the characters without cluttering the main event loop.
3) Implemented forever loops to ensure persistent monitoring of the in-game variables.
